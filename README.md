# 🤖 AI Text Analyzer

A simple AI-style text analyzer that summarizes input and extracts key points.

## 🚀 Features
- Text summarization
- Keyword extraction
- Runs in terminal (no API required)

## 🛠️ Tech Stack
- Python

## ▶️ How to Run

1. Open terminal
2. Navigate to project folder
3. Run:
python ai_text_analyzer.py
